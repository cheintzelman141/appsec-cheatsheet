# appsec-cheatsheet

A living Application Security + Cybersecurity reference maintained by a senior software engineer transitioning into AppSec.

This repo contains:
- Web AppSec attack patterns & remediation notes (OWASP focus)
- Recon / enumeration command references
- Tooling quick-use guides (Burp workflows, nmap, ffuf, etc.)
- Defensive fundamentals (SOC, IR, logging, SIEM basics)
- Malware/forensics references (CAPA, REMnux, Volatility, FLARE VM)

## How to use
- Browse `/docs` by topic
- Use as a quick reference during labs (TryHackMe) and security-focused dev work
- Pair with my secure coding demos (see pinned repos)
